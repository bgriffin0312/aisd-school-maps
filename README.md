# Austin ISD Attendance Boundaries — Illustrative Scenarios

An interactive map for comparing Austin ISD attendance-zone boundary sets at the
elementary, middle, and high school levels.

**These are illustrative scenarios, not a proposed boundary map.** They are
analytical examples — existence proofs — showing what is achievable under
different priorities (for example, deconcentrating economic disadvantage). They
are intended to inform a policy conversation, not to advocate a specific set of
boundaries for adoption.

## Boundary sets included

- **Current boundaries** — Austin ISD's existing attendance zones (pre-redraw).
- **Scenario 14** — Austin ISD's own draft redraw.
- **Illustrative scenario** — an analytical example that prioritizes reducing
  concentrated economic disadvantage, used to show what is achievable. It is an
  existence proof, not a proposed boundary plan.

The map can highlight which block groups change between any two sets.

## Data sources

Austin ISD attendance-boundary shapefiles; U.S. Census American Community Survey
(ACS) 2018–22; Texas Education Agency (TEA) 2024–25. Built with
[Leaflet](https://leafletjs.com/); basemap © OpenStreetMap, © CARTO.

## Viewing

Open `index.html` (served over HTTP — e.g., via GitHub Pages). The page fetches
the GeoJSON files alongside it, so it must be served, not opened as a local file.
